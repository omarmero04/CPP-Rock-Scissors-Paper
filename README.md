# CPP-Rock-Scissors-Paper
A command-line implementation of the classic Rock, Scissors, Paper game developed in C++. This project supports two modes of play: Player vs. Player and Player vs. Computer. The computer mode utilises the cstdlib and ctime libraries for true random choice generation.

This is an interactive, multi-mode version of the classic Rock, Scissors, Paper game implemented entirely in C++. It offers players the flexibility to compete against a friend (local two-player mode) or test their luck against a randomly selected computer opponent.
